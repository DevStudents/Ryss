# Ryss
Ryss is programming language with syntax inspired by Assembly.

### Simple "Hello World" program:
#### First option:
```
log 'Hello World!'
```

#### Second option:
```
init firstVariable 'Hello World!'
log firstVariable
```
Output in both cases is the same ```Hello World!```
